# TerminalMarkDown

## Members
- Z (Ziang) Xiao

TerminalMarkDown takes a MarkDown file from the user, and displays it in preview mode in a terminal tab.

Example Usage
```terminalMarkDown example.md```

## Basic
- Emphasis (italics, bold, underline)
- Background Highlight
- Code Blocks
- Headers
- Nested Lists
- Links
